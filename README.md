# Reshme-Namma Pride (ರೇಷ್ಮೆ ನಮ್ಮ ಹೆಮ್ಮೆ)

A smart climate monitoring app for silkworm rearing where farmers log temperature and humidity 3× daily and receive SAFE/CAUTION/DANGER advice powered by Google Gemini AI, with a full offline fallback using KSRDI sericulture data. Built with Kotlin (Android) and Next.js 15 + Firebase (Web). VTU BINT803B Internship 2025–26.
